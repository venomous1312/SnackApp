# SnackApp 🍎
**A Full-Stack Delivery Management Platform**

### 📝 Project Overview
SnackApp is a custom-built web application designed to streamline local delivery logistics. Built with the **Django** framework, it features a multi-portal architecture that allows business owners to manage inventory while providing drivers with a dedicated interface for order tracking.

### 🛠 Technical Stack
* **Backend:** Python & Django
* **Database:** SQLite (Development) / PostgreSQL (Ready)
* **Frontend:** HTML5, CSS3, JavaScript, Django Template Language
* **Tools:** Cursor AI (optimized for high-velocity refactoring)

### 🚀 Key Features
* **User Role Logic:** Custom authentication for distinct Business, Driver, and Customer roles.
* **Order Management System:** A full CRUD (Create, Read, Update, Delete) cycle for tracking deliveries from order placement to final completion.
* **Driver Portal:** Responsive dashboard for drivers to accept and update order statuses in real-time.
* **Data Integrity:** Robust Django models to ensure relational data between orders, inventory, and users.

### 🏗️ How to Run Locally
1. Clone the repository: `git clone https://github.com/your-username/SnackApp.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run migrations: `python manage.py migrate`
4. Start server: `python manage.py runserver`
